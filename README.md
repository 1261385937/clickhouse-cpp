ClickHouse C++ client
=====

[![Build Status](https://travis-ci.org/artpaul/clickhouse-cpp.svg?branch=master)](https://travis-ci.org/artpaul/clickhouse-cpp)

## OVERVIEW

C++ client for [Yandex ClickHouse](https://clickhouse.yandex/)

## BUILDING

```
$ mkdir build .
$ cd build
$ cmake ..
$ make
```
